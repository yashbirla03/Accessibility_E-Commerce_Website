# Mobile and Tablet Accessibility Implementation Map

| Guideline Name | Category | Implemented Screen(s) |
| :--- | :--- | :--- |
| **1. Offer alternatives to gestures** | Physical / Mobility | All screens (Carousels, Sliders) |
| **2. Support larger text sizes** | Vision | Global CSS / All screens |
| **3. Adaptable (Reflow – single column layout on mobile)** | Vision / Cognitive | Global CSS / Index & Product Listing |
| **4. Text spacing** | Cognitive / Vision | Global CSS |
| **5. Target size (sufficiently sized controls)** | Physical / Mobility | Global CSS / Navigation & Checkout |
| **6. Concurrent input mechanisms (voice integration)** | Physical / Cognitive | Header / Search |
| **7. Change on request (minimize time-based UI)** | Cognitive | Checkout / Modals |
